虽然目前是入门级前端开发实习生，但是出于对技术的好奇，还是每天花了一些时间把Python的基础语法看了一遍。决定以后把Web前端作为主业，Python作为副业。早听说Python可以做任何事情，而且在人工智能领域有卓越的表现，因此通过分析一些案例和写一些简单的代码来提升对Python的理解以及提升编码能力。

爬虫常规思路

  1. 明确目的，爬取熊猫TV 网站英雄联盟游戏分类下面所有主播的人气排行
  2. 使用Chorme浏览器的开发者工具找到需要爬取数据的位置和标签。
  3. 模拟HTTP请求，向服务器发送这个请求，获取服务器返回给我们的HTML
  4. 用正则表达式提取需要的数据
  5. 对数据进行一些业务逻辑的处理

数据提取分析的三个原则

  1. 尽量选取唯一标识性的标签作为定位标签
  2. 尽量选取最贴近数据的标签
  3. 选取标签的时候，尽量选取可以闭合的标签（兄弟标签会使匹配变得复杂）
