Tree-shaking - JS Tree-shaking

如果有一些代码不再用到或从来不用，会造成资源的浪费，有一部分资源从来没有用到。

Tree-shaking分为两种：

  • JS Tree Shaking (把多余方法和文件去掉)
  • CSS Tree Shaking (有些样式从来都匹配不上，这些样式不应该打包)

使用场景

  • 常规优化
  • 引入第三方库的一个功能 (比如lodash中的一两个功能、boostrap的一两个UI)
